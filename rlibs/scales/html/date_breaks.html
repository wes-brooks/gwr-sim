date_breaks {scales}R Documentation

Regularly spaced dates.

Description

Regularly spaced dates.

Usage

  date_breaks(width = "1 month")

Arguments

width

an interval specification, one of "sec", "min", "hour", "day", "week", "month", "year". Can be by an integer and a space, or followed by "s".
