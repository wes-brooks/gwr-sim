Mute standard colour.

Description

Mute standard colour.

Usage

  muted(colour, l = 30, c = 70)

Arguments

colour

character vector of colours to modify

l

new luminance

c

new chroma

Examples

muted("red")
muted("blue")
show_col(c("red", "blue", muted("red"), muted("blue")))
