dev_example {devtools}R Documentation

Run a examples for an in-development function.

Description

Run a examples for an in-development function.

Usage

  dev_example(topic)

Arguments

topic

Name or topic (or name of Rd) file to run examples for

See Also

Other example functions: run_examples

Examples

## Not run: 
# Runs installed example:
library("ggplot2")
example("ggplot")

# Runs develoment example:
load_all("ggplot2")
dev_example("ggplot")

## End(Not run)
