adalasso-package {adalasso}R Documentation

Use the adaptive lasso for shrinkage and selection in a generalized linear model

Description

Iterate between glmnet (for shrinkage/selection) and glm (for adaptive penalty weights) to get an adaptive lasso estimate of coefficients in a GLM where p>n.
