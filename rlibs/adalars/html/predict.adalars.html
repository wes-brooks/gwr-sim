predict.adalars {adalars}R Documentation

Make predictions using a adalars object

Usage

predict.adalars(object, newx)

Arguments

object

The adalars object to use for prediction.

newx

Data frame to use in prediction.

Examples

##---- Should be DIRECTLY executable !! ----
##-- ==>  Define data, use random,
##--	or do  help(data=index)  for the standard data sets.

## The function is currently defined as
function(object, newx) {
    pred.data = as.matrix(newx[,object[['lars']][['vars']]])
    return(predict.lars(object[['lars']][['model']], newx=pred.data, type='fit', s=object[['lars']][['lambda.index']], mode='step')$fit)
}
